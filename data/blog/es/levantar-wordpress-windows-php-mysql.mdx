1. **Instalar PHP:**
   - Descarga e instala PHP en tu sistema si aún no lo tienes. Puedes obtener la última versión desde el sitio web oficial de PHP (https://www.php.net/downloads).
   - Asegúrate de configurar PHP para que se ejecute desde la línea de comandos siguiendo las instrucciones de instalación.

2. **Descargar e Instalar WordPress:**
   - Descarga la última versión de WordPress desde su sitio web oficial (https://wordpress.org/).
   - Extrae el archivo ZIP en el directorio de tu elección en tu sistema.

3. **Configurar una Base de Datos:**
   - Utiliza una herramienta de administración de bases de datos como phpMyAdmin o MySQL Workbench para crear una nueva base de datos MySQL.
   - Toma nota de los detalles de la base de datos, como el nombre de la base de datos, el nombre de usuario y la contraseña.

4. **Configurar el Archivo de Configuración de WordPress:**
   - Encuentra el archivo `wp-config-sample.php` en el directorio de WordPress y cámbiale el nombre a `wp-config.php`.
   - Abre `wp-config.php` en un editor de texto y actualiza los detalles de la base de datos con los que has creado anteriormente.

5. **Iniciar el Servidor PHP:**
   - Abre una terminal o línea de comandos y navega hasta el directorio donde has instalado WordPress.
   - Ejecuta el siguiente comando para iniciar un servidor PHP incorporado:
     ```
     php -S localhost:8000
     ```
   - Esto pondrá en marcha un servidor PHP en el puerto 8000 de tu máquina local.

6. **Instalar WordPress:**
   - Abre un navegador web y ve a `http://localhost:8000`.
   - Sigue las instrucciones en pantalla para completar la instalación de WordPress, proporcionando los detalles de la base de datos cuando se te soliciten.

Con estos pasos, tendrás un servidor de WordPress en funcionamiento utilizando PHP, similar a cómo se haría en un entorno de desarrollo con Laravel. A partir de aquí, puedes acceder al panel de administración de WordPress y comenzar a configurar tu sitio web según tus necesidades.
