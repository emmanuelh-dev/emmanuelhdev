## Otros Usos del Sensor de Temperatura

Además de monitorear la temperatura ambiente, el sensor de temperatura del Coban 405 también puede configurarse para enviar alertas si la temperatura sale de un rango preestablecido.

### Configurar Alertas de Temperatura

Puedes activar las alertas de temperatura alta y baja enviando el siguiente comando SMS:

```
temperature+contraseña+espacio+temperatura_baja,temperatura_alta
```

Por ejemplo:

```
temperature123456 -10,40
```

Este comando establecerá una alerta si la temperatura cae por debajo de -10°C o sube por encima de 40°C.

Cuando se activa una alerta de temperatura, el rastreador:

- Envía un SMS de alerta a los números autorizados
- Hace sonar la sirena por 20 segundos

Para desactivar las alertas de temperatura:

```
notemperature+contraseña
```

Esta función de alertas por rango de temperatura puede ser útil para monitorear condiciones extremas, carga sensible al calor/frío, etc.

## Instalación del Sensor de Temperatura

El manual indica que el sensor de temperatura es una opción adicional que puede venir incluida o no, dependiendo del modelo/paquete del Coban 405 adquirido.

Si tu unidad no cuenta con el sensor de temperatura, puedes adquirirlo por separado o consultarle al proveedor la opción de instalarlo de fábrica.

Una vez instalado el hardware del sensor, podrás acceder a todas las funciones de monitoreo de temperatura y configuración de alertas mediante los comandos SMS descritos.

## Lista de Comandos SMS para configurar el GPS coban (Aplica para cualquier version del mismo)

Para configurar un GPS Coban requieres enviar comandos SMS con la contraseña. La contraseña por defecto es 123456.

Monitoreo de Temperatura en Modo GPRS:

```
protocol123456 18
```

Activa el protocolo extendido que incluye temperatura en los reportes GPRS.

Alertas de Temperatura:

```
temperature123456 -10,40
```

Configura una alerta si la temperatura cae por debajo de -10°C o sube por encima de 40°C.

```
notemperature123456
```

Desactiva las alertas de temperatura.

Verificar Temperatura:

```
check123456
```

Solicita el estado del dispositivo, que incluye la temperatura actual entre otros datos.

Obtener Ubicación y Temperatura por SMS:

```
fix090s***n123456
```

Solicita un reporte de ubicación.

```
check123456
```

Solicita el estado con temperatura.

Además, puedes configurar contraseña, APN, modo de rastreo, alertas diversas y más con comandos como:

```
password123456 nueva_contraseña
apn123456 APN_de_tu_operador
sleep123456 off 030s 030m 005n
stockade123456 22.554,113.903;22.553,113.909 //Geocerca
```

Consulta el manual de usuario para conocer todos los comandos disponibles. Es recomendable cambiar la contraseña por defecto por una nueva después de la instalación inicial.
